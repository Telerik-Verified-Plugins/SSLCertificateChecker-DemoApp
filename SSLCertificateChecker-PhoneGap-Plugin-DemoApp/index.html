SSLCertificateChecker Plugin Demo

The SSLCertificateChecker plugin allows you to perform Certificate Pinning in order to prevent Man in the Middle (MitM) attacks.

You may want to check the connection when the app is started, but you can choose to invoke this plugin every time you communicate with the server. In either case, you can add your logic to the success and error callbacks.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android - App package.
